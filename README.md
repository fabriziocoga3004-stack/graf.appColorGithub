#	Instituto Tecnologico de Pachuca 
## Materia: Graficacion 
## Tema 1.2 Publicacion colavorrativos de codigo fuente 
## Autor: Alberto Fabrizio Cortes Garcia
## Fecha: 4/Febrero/2025
La aplicación ofrece varias maneras de elegir un color desde controles
deslizantes, campos numéricos y un selector de color, todos estos están
conectados ya que al modificar alguna de las secciones se actualiza el color, así
como también los valores RGB, HEX.
Como dato se organizó en archivos diferentes para hacerlo mas comprensible el
archivo HTML se encargó de la estructura, el CSS del diseño visual y el
JavaScript del funcionamiento de la aplicación esto hizo que el proyecto
fuera mas sencillo de modificar, además, se logró un diseño moderno y
atractivo que mejora la experiencia del usuario, haciendo que la
aplicación sea más agradable y fácil de usar.
